# Mac 软件

**需破解的软件，必须存储到百度网盘、腾讯微云等平台进行备份**

```sh
# 已损坏,⽆法打开。 您应该将它移到废纸篓。
sudo spctl --master-disable
sudo xattr -d com.apple.quarantine /Applications/xxx.app
```
